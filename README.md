# Chip8 JS emulator

A chip8 emulator, an 8 bit language from the '70s, re-written in Javascript.


# Controls
| KEYBOARD |\
| 1 | 2 | | 3 | | 4 |\
| Q | W | | E | | R |\
| A | S | | D | | F |\
| Z | X | | C | | V |

# Roms
It can emulate all chip8 roms, in the repo there's 3:

 - BREAKOUT
 - SOCCER
 - BLINKY1
